# Отчет по проекту "Проектный практикум 1"

## Проект
Backend для учебной платформы: курсы, уроки, тестирование, отзывы.

## Роль
Разработчик. Отвечал за модуль отзывов к курсам.

## Что было сделано 
Реализовал функционал отзывов к курсам:

- Получение списка отзывов по курсу 
- Создание отзыва 
- Обновление отзыва 
- Удаление отзыва (архивация) 

Дополнительно:
- Добавлена проверка существования курса при обращении к отзывам по `course_id`
- Учитывается флаг `archived`: архивные отзывы не попадают в выдачу


## Технологии и инструменты
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Docker
- Redis
- Git 
- Swagger 

## Итоги
Модуль отзывов реализован и проверен, архивирование учитывается,в выдаче отзывов по курсу тоже все гуд.
