## Индивидуальный отчет по проекту в дисциплине "Проектный практикум 1"

### Проект

Приложение для обучения с возможностью просмотра курсов,
прохождения уроков, тестирований, отслеживания прогресса и написания отзывов.

### Роль

В проекте я занимал роль тимлида-разработчика, а также отвечал за написание модулей уроков и прогресса по курсам.

### Задачи

Главной моей задачей было создание создание модулей `Lesson` и `UserCourse`. В нее вошли следующие этапы:

- Проектирование моделей данных `Lesson` и `UserCourse`
- Реализация слоев `LessonRepository` и `UserCourseRepository` для доступа к данным
- Разработка Pydantic-схем валидации данных `LessonSchema` и `UserCourseSchema` и сервисных слоев `LessonService` и `UserCourseService`
- Интеграция с модулями `Course` и `TestQuestion` и настройка зависимостей
- Разработка API
  Эндпоинты для `Lesson`:
  - Получение урока по id и списка уроков для конкретного курса
  - CRUD-операции

  Эндпоинты для `UserCourse`:
  - Получение краткого прогресса пользователя по курсу
  - Получение расширенного прогресса пользователя по курсу, включающего в себя список вопросов, на которые пользователь дал ответ, а также прогресс по урокам
  - Получение расширенного прогресса пользователя по уроку.
  - Запись пользователя на курс
  - Ручное обновление ответа пользователя на конкретный вопрос
  - Сброс прогресса пользователя для конкретного урока 
  - Мягкое удаление блока общего прогресса
