## Отчет по проекту в дисциплине "Проектный практикум 1"

### Роль

В проекте "Учебная платформа" моей основной ролью была тестировщик, также я написала несколько эндпоинтов для модуля курсы.

### Задачи

Главной моей задачей было тестирование эндпоинтов проекта с помощью Pytest, которые помогли выявить и исправить неточности в работе учебной платформы. 

Каждый тест гарантированно работает с очищенной БД, создание данных для тестов происходит автоматически. В тестах проверяются граничные условия, корректная валидация данных, ограниченность доступа для определенных ролей пользователей.

Для модуля курсы я написала следующие эндпоинты:

- создание нового курса
- обновление курса
- удаление курса


### Технологии и инструменты

- Python
- Pytest
- FastAPI
- Pydantic
- UV
- SQLAlchemy
- DBeaver
- Docker
- Git

### Итоги

Получилось покрыть тестами большую часть проекта. Все написанные тесты работают корректно.
Благодаря тестам получилось выявить несколько багов в проекте, которые разработчики успешно исправили.

**Основные трудности**

Для тестирования проекта необходимо было вникнуть в каждый модуль, выяснить для чего используется эндпоинт, узнать входные и выходные данные при корректной работе и в случае ошибок. 
Также надо было знать корректную работу эндпоинта, чтобы тест проверял верную логику, а не был подстроен под написанный код.
Для написания тестов необходимо автоматизированно создавать данные, что дополнительно включает в себя работу с БД.
