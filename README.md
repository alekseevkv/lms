## Итоговый проект по дисциплине "Язык Python для разработчиков"

### Учебная платформа

Приложение для обучения с возможностью просмотра курсов,
прохождения уроков и тестирований.

На главной странице находится каталог курсов, он доступен без авторизации. После регистрации и авторизации у пользователя появляется возможность записаться на курс, читать или смотреть уроки, проходить тесты к урокам. Также есть возможность оставить отзыв о курсе.

Есть два вида регистрации: в качестве студента и админа. У админа есть возможность создавать и редактировать пользователей, курсы, уроки и тестовые вопросы. Обычные пользователи могут редактировать только свои профиль и отзывы.

Запуск dev контейнеров:
```bash
docker compose -f docker-compose.dev.yml up --build
```

Установка локальных зависимостей:
```bash
uv sync
```

Запуск миграций:
```bash
uv run alembic upgrade head
```

Запуск тестов:
```bash
 uv run pytest tests
```
